Meetups

October 24th, 2017

Meetup Information

The first meetup I went to was the DenverScript meetup on October 24th, at CodeTalent. The meeting covered Chrome Dev tools (Chrome 60) and CSS Variables.

People I Met

I met James Sullivan, a recent Turing Web Dev graduate. James is on the job hunt and optimistically so. James cited the meetups as a great place to network, pick up new skills and score free meals.

I first met Guillermo Moratorio on the 4th floor Platte balcony and then again at this meetup. Guillermo is a Galvanize graduate and works as software developer at Voalte. Super nice guy!

What I Learned

I learned that CSS variables (custom properties) permit the declaration and usage of cascading variables in stylesheets. This is considered experimental but is 78% globally supported.

I learned that I was able to understand the syntax in the lightning talks for the most part.

I learned that the Chrome dev tools are an extremely helpful tool that is constantly being updated. The talk was on Chrome 60 and 61 has already been released. The updates make it easier to work directly in the console when debugging and editing code.

Objects in the console now display more information than before.

Questions I Have

  • Will we be learning about CSS Variables at Galvanize/is it a "game changer"?
  • Is it best practice to use the dev tools for all debugging etc.?
  • The dev tools have SO MANY different menus etc., what are the main functions to learn?
